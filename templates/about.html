ABOUT THE PLATFORM

Intelligent X-ray Contraband Detection

Airport Security AI is an intelligent X-ray baggage screening platform designed for automatic detection of prohibited objects using custom-trained YOLO object detection models.

PROJECT OVERVIEW

AI-Powered Security Screening

The platform combines deep learning object detection with an interactive web application for X-ray baggage image analysis.

◈

X-ray Image Analysis

Users can upload baggage X-ray images for automatic prohibited-object detection.

◎

Object Detection

The system uses a trained YOLO detection model to locate and classify detected objects.

◇

Model Analytics

The Analytics dashboard provides performance comparisons across the evaluated YOLO models.

◫

Detection History

Detection results are recorded so previous screening activity can be reviewed.
